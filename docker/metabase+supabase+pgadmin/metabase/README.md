# Metabase assets

## Plugins

Coloque JARs de plugins em `metabase/plugins/`.

> Reinicie o serviço `metabase` após adicionar/atualizar plugins.

## Dashboards (serialização)

Use `metabase/dashboards/` para versionar exports/imports de conteúdo do Metabase.

- Export manual via UI/API
- Import via script: `../metabase-import.sh`
